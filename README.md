## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

```bash
cd hesi-app
npm install
```

### Development

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

### Build for Production

```bash
npm run build
```

## Deployment

This app is designed to deploy to GitHub Pages:

```bash
npm run build
# Deploy the dist folder to gh-pages branch
```

## Content Source

Study content is extracted from the HESI Admission Assessment Exam Review (5th Edition) textbook. All practice problems include verified answers from the answer key.
